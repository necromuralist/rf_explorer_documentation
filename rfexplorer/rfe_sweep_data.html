RFE Sweep Data¶

The RFESweepData holds the data for a single sweep of the spectrum (I think).

RFESweepData(fStartFreqMHZ, fStepFreqMHZ, …) Class support a full sweep of data from RF Explorer, and it is used in the RFESweepDataCollection container

Get The Peak¶

The first example given by the code uses the Peak Step to print out some information. The Peak Step is the frequency that had the largest amplitude for that sweep. The GetPeakStep will return the index (step) that matches the peak which you can then use to make some queries.

RFESweepData.GetPeakStep() Returns the step of the highest amplitude value found
RFESweepData.GetAmplitude_DBM(nStep) Returns amplitude data in dBm.
RFESweepData.GetFrequencyMHZ(nStep) Get frequency in MHz 
peak_step = sweep_data.GetPeakStep()
peak_amplitude = sweep_data.GetAmplitude_DBM(peak_step)
peak_frequency = sweep_data.GetFrequencyMHZ(peak_step)

CSV Output¶

These methods will dump the sweep data with a (quasi) CSV-format.

RFESweepData.Dump() Dump a CSV string line with sweep data.
RFESweepData.SaveFileCSV(sFilename, …) Save a CSV file using one frequency point/dBm value per line

Note

When the RFESweepData is instantiated it sets a datetime.datetime.now() value in an attribute named self.m_Time which might be useful when constructing your own CSV. Or use RFESweepData.CaptureTime which is the same thing.
